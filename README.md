个人简历

这是一个开源的个人简历项目.工作之余使用vue练练手。大家可以在本项目中不仅仅可以学习了解以下的框架/库，同时还可以了解关于代码结构组织,模块化,前端构建等内容。这个项目才刚开始，我会带着大家完成整个项目的开发工作。

技术栈

前端
es6
vue2
webpack2
vue-router2
axios

后端
node
mongoDB

起手式

  node -v
  v6.9.2

  npm -v
  3.10.9
  1. fork本项目

  2. git clone ...

  3. cd vqlai.github.io

  4. npm install （建议使用淘宝镜像）

  5. npm run dev

  6. 浏览器打开 localhost:1060
